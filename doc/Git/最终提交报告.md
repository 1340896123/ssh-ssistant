## 设计变更详解

### 颜色方案变更

#### 原方案（北极光风格）
- 主色调：蓝色渐变（#4F46E5 → #7C3AED → #06B6D4）
- 背景：深色（#0A1628 → #0F1728）
- 强调色：霓虹发光效果
- 特点：渐变、发光、动画

#### 新方案（浅色简约）
- 主色调：深墨灰 #2B2B2B
- 强调色：铁锈红 #C75B5B
- 背景：纯白 #FFFFFF → 浅灰 #F8F8F8
- 特点：扁平、静态、专业

### 字体系统
- **UI字体**：Inter（400, 500, 600, 700）
- **代码字体**：JetBrains Mono（400, 500）
- **加载方式**：Google Fonts CDN


## 技术实现

### 组件改造模式

每个组件都遵循以下改造模式：

1. **移除硬编码颜色**
   - gray-* → bg-bg-* / text-text-*
   - blue-* → accent
   - green-* → success
   - red-* → error
   - yellow-* → warning

2. **移除未定义的样式类**
   - glass-* → bg-bg-elevated
   - shadow-glow → shadow-md
   - glow-border → border-border-primary

3. **统一使用CSS变量**
   - 所有颜色通过CSS变量定义
   - 统一的语义化命名
   - 便于主题切换

### 性能优化

1. **减少运行时计算**
   - 移除GPU密集型动画
   - 减少box-shadow的复杂度
   - 简化transition效果

2. **减少代码量**
   - 删除未使用的CSS
   - 简化样式定义
   - 提高可维护性
