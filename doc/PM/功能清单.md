# SSH Assistant 产品功能清单

## 产品概述
SSH Assistant 是一款基于 Tauri + Vue 3 + TypeScript 构建的现代化 SSH 桌面客户端，集成了强大的终端功能、AI智能助手和完整的文件管理器，旨在提升远程服务器管理效率。

**目标用户群体**: 开发者、系统管理员、DevOps工程师、运维人员

---

## 核心功能模块

### 1. 连接管理模块
**功能清单**:
- SSH连接创建、编辑、删除
- 密码认证支持
- Jump Host/Bastion（跳板机）连接支持
- 本地SQLite持久化存储
- 连接状态实时监控

**设计风格影响**: 无功能变更，仅需UI样式调整

### 2. 会话管理模块
**功能清单**:
- 多标签页支持，管理多个服务器会话
- 会话间完全隔离
- 每个会话集成终端、文件管理、AI助手
- 会话持续时间统计
- 会话重连/断开功能

**设计风格影响**: 无功能变更，仅需UI样式调整

### 3. 智能终端模块
**功能清单**:
- 基于xterm.js的全功能终端
- 传统Tab自动补全
- **AI智能补全**: 上下文感知的智能命令建议
- 终端内容搜索功能
- 命令历史记录
- Zmodem文件传输协议支持

**设计风格影响**: 无功能变更，仅需UI样式调整

### 4. 文件管理模块

**功能清单**:

- 远程文件浏览、上传、下载
- 拖拽上传/下载支持
- 文件/目录创建、删除、重命名
- 远程文件本地编辑（自动下载→监控变更→自动上传）
- 断点续传和文件完整性校验
- 虚拟滚动（处理大目录）
- 双布局模式（底部布局/左侧布局）- 支持持久化存储
- 视图模式切换（平铺/树形）- 支持持久化存储
- SFTP缓冲区大小配置 - 支持持久化存储

**设计风格影响**: 无功能变更，仅需UI样式调整

**功能配置项**:

| 配置项 | 类型 | 默认值 | 持久化状态 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| viewMode | flat/tree | flat | ✅ 已持久化 | 文件浏览视图模式 |
| layout | left/bottom | bottom | ⚠️ 待持久化 | 文件管理器布局位置 |
| sftpBufferSize | number (KB) | 512 | ✅ 已持久化 | SFTP传输缓冲区大小 |

**待优化功能**:

- [ ] layout 字段持久化存储（当前仅前端内存存储，重启后丢失用户配置）

### 5. AI助手模块
**功能清单**:
- 上下文感知的智能对话
- 聊天界面直接执行终端命令
- 可配置AI模型参数（API地址、密钥、模型名称）
- 工具调用支持
- 实时命令输出显示
- 会话历史管理

**设计风格影响**: 无功能变更，仅需UI样式调整

### 6. 个性化设置模块
**功能清单**:
- 多语言支持（中文/English）
- UI主题切换
- 自定义AI补全和聊天配置
- 文件管理器布局配置
- 终端字体和配色设置

**设计风格影响**: 无功能变更，仅需UI样式调整

### 7. 系统监控模块
**功能清单**:
- 实时CPU使用率监控
- 内存使用率监控
- 磁盘使用率监控（多挂载点）
- 进程信息展示（Top 5进程）
- 服务器IP地址显示
- 系统运行时间显示

**设计风格影响**: 无功能变更，仅需UI样式调整

### 8. 通知系统模块
**功能清单**:
- 全局通知弹窗
- 成功/错误/警告消息类型
- 自动关闭机制
- 操作反馈提示

**设计风格影响**: 无功能变更，仅需UI样式调整

## 设计风格变更说明

### 当前设计风格: Arctic Glow 北极光风格
**特点**:
- 炫酷科技感视觉风格
- 渐变极光色彩效果
- 发光边框和阴影效果
- 高饱和度配色
- 动画过渡效果较多

### 新设计风格: Minimalist Light 浅色简约风格
**特点**:
- 简洁专业的视觉风格
- 纯净的单色背景
- 细腻的边框和阴影
- 低饱和度配色
- 轻量级动画效果

---

## 配色方案定义

### Minimalist Light 配色体系

| 色彩类型 | 色值 | 应用场景 |
|---------|------|---------|
| **基底色** | `#FFFFFF` 纯白 | 主背景色 |
| **次级背景** | `#F8F9FA` 浅灰 | 次级背景、卡片背景 |
| **三级背景** | `#F1F3F5` 中灰 | 输入框、下拉框背景 |
| **主色调** | `#2B2B2B` 深墨灰 | 主要文字、标题 |
| **次要文字** | `#6C757D` 中灰 | 次要文字、说明文字 |
| **边框色** | `#DEE2E6` 浅灰边框 | 分隔线、边框 |
| **强调色** | `#C75B5B` 铁锈红 | 按钮、高亮、重要操作 |
| **成功色** | `#40916C` 森林绿 | 成功状态、在线状态 |
| **警告色** | `#F4A261` 沙漠橙 | 警告状态、中等负载 |
| **错误色** | `#E63946` 珊瑚红 | 错误状态、高负载 |
| **悬停色** | `#E9ECEF` 浅灰悬停 | 按钮/元素悬停状态 |

### 字体定义
| 字体类型 | 字体家族 | 应用场景 |
|---------|---------|---------|
| **UI字体** | `Inter, -apple-system, BlinkMacSystemFont, sans-serif` | 界面文字 |
| **代码字体** | `JetBrains Mono, Consolas, monospace` | 终端、代码、命令 |

## 用户价值提升

### 性能提升
- 移除复杂的渐变和发光效果，减少渲染负担
- 简化CSS动画，提升界面响应速度
- 降低GPU使用率，延长电池续航（笔记本电脑）

### 易读性提升
- 高对比度的黑白配色，文字更清晰
- 统一的字体系统（Inter + JetBrains Mono）
- 充足的留白，信息层次更分明
- 减少视觉干扰，提升专注度

### 专业感提升
- 简约设计更符合专业工具定位
- 与主流IDE/编辑器（VS Code、JetBrains）风格统一
- 适合长时间工作使用，视觉疲劳度低
- 企业级应用的视觉形象

### 可维护性提升
- 配色方案简单，主题切换更容易
- 组件样式复用性高
- 未来扩展功能时设计一致性强
