# SSH远程连接助手
traui+ vite +vue3 + ts+ tailwindcss4.1

## 核心功能模块

- **本地数据存储**：支持保存连接管理信息以及配置管理信息到sqllite数据库中持久化
- **连接管理**：支持SSH连接的创建、编辑和删除，创建会话,仅支持密码认证，名称，主机，端口，用户名，密码
- **会话管理**：多会话并发支持，采用标签页界面展示，一个会话包含一个文件管理模块，一个终端，一个AI助手模块，会话状态展示
- **配置管理**：系统配置和用户偏好设置，中文，英文语言切换，风格主题切换，AI会话配置和命令补全配置支持自定义API基础URL、API密钥和模型名称
- **文件管理**：远程文件浏览、上传、下载，支持目录文件创建、删除、重命名，复制路径，复制名称，双击下载文件到本地目录后自动通过本地程序打开，保存后自动上传到服务器
- **终端交互**：支持持续交互终端命令以及支持传统自动补全功能，以及AI自动补全功能
- **AI助手**：智能对话辅助，可执行终端命令完成用户任务，无需设置命令执行权限

## 会话隔离机制
- 支持多会话并发操作，通过标签页界面展示
- 各会话间完全隔离，即使是同一连接创建的多个会话
- 文件管理、终端操作和AI助手功能不能单独存在，必须在会话链接后才会出现。
- 一个会话只有一个终端

## 技术架构
- **后端**：Rust + Tauri
- **前端**：TypeScript + Vite + Vue.js
- **样式**：TailwindCSS 4.1
- **要求**：界面布局合理，用户体验流畅

## AI助手配置
- 支持自定义API基础URL
- 支持自定义API密钥
- 支持自定义模型名称
